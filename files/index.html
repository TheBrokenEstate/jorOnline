Welcome to jorOnline

Grab a seat, make yourself at home, nice to meet you. jorOnline is a website that uses websockets to let you talk to people from anywhere at any time.

Commands!

All commands beginning with a '/' must be used at the beginning of a message.

  • @user # Calls user with notification.
  • *message* # Makes things italic!
  • http://link.ext # Hyperlink to website.
  • https://link.ext # More secure hyperlink (Make sure website supports it).
  • /c colour message # Changes colour of your message.
  • /uc colour # Changes colour of your username.
  • /me action # Perform an action as yourself.
