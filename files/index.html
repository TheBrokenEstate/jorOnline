File and Image Sharing

Newly added to jorOnline is file and image sharing, you can select a file with the browse button and send. Below is a list of supported file formats. All file formats have to be tested to comply with our method of transfer and we can only currently support up to 15MB files. We are sorry if we don't currently support what you need.

Images:

  • png
  • jpg
  • jpeg
  • bmp
  • gif

Text Format:

  • txt
  • py/pyw
  • c
  • css
  • js

Downloads:

  • zip
  • iso
  • exe
  • msi
  • pyc


Commands!

All commands begining with a '/' must be used at the begining of a message.

  • @user # Calls user with notification.
  • *message* # Makes things italic!
  • http://link.ext # Hyperlink to website.
  • https://link.ext # More secure hyperlink (Make sure website supports it).
  • /uc colour message # Changes colour of your username.
  • /c colour message # Changes colour of your message.
  • /me action # Perform an action as jorself.
  • /i quote # Quote the Ianster.
  • /uwu # They made me add this...
